# Structured Interview Evaluator

**Category**: Executive & Business
**Difficulty**: Intermediate
**Source**: [HR Interview Transcription Guide](https://brasstranscripts.com/blog/hr-interview-transcription-hiring-guide-2026)

## Description

Analyzes job interview transcripts to create structured evaluations for hiring committees. Extracts candidate responses, applies STAR methodology analysis, and generates evidence-based scoring recommendations for each competency.

## Use Cases

- Final-round hiring decisions requiring documentation
- Panel interview evaluations with multiple reviewers
- Behavioral interview scoring using STAR methodology
- Compliance documentation for EEOC requirements
- Structured interview program implementation

## The Prompt

```text
Analyze this job interview transcript and create a structured evaluation for the hiring committee.

INTERVIEW CONTEXT:
- Position: [Job title]
- Key competencies to evaluate: [List 3-5 job-related competencies]
- Interview format: [Behavioral/Technical/Case study/Panel]

For each competency listed, extract and evaluate:

1. **Competency: [Name]**
   - Relevant candidate statements (quote directly from transcript)
   - STAR analysis if applicable:
     * Situation described
     * Task/responsibility
     * Action taken
     * Result achieved
   - Suggested score (1-5): [Score]
   - Scoring rationale: [Why this score based on evidence]

2. **Overall Candidate Assessment**
   - Strongest competencies demonstrated
   - Areas requiring further evaluation
   - Red flags or concerns (if any)
   - Recommended next steps

3. **Documentation Notes**
   - Job-related observations only (exclude protected characteristics)
   - Specific examples that support or contradict job fit
   - Questions for reference check (if applicable)

CRITICAL: Base ALL evaluations on specific statements from the transcript. Do not infer or assume information not explicitly stated by the candidate.

TRANSCRIPT:
[Paste your interview transcript here]

---
```

## Expected Output

The prompt generates a structured evaluation document including:
- Competency-by-competency analysis with evidence
- STAR methodology breakdown for behavioral responses
- Numerical scoring recommendations with rationale
- Overall assessment and hiring recommendation
- Compliance-appropriate documentation notes

## Customization Tips

- Add specific competencies relevant to your role (e.g., "Technical Problem Solving", "Leadership", "Customer Orientation")
- Adjust the scoring scale to match your organization's rubric
- Include role-specific evaluation criteria for technical positions
- Add prompts for culture-fit assessment (using job-related criteria only)
