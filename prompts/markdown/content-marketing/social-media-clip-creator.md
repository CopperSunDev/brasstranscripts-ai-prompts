# Social Media Clip Creator

**Category**: Content Marketing
**Difficulty**: Advanced
**Estimated Tokens**: 800-1200
**Version**: 1.0.0

## Description

Transform identified highlights into optimized social media content with platform-specific formatting. This prompt creates tailored content strategies for TikTok, YouTube Shorts, Instagram Reels, and Twitter.

## The Prompt

```text
Please take these highlight moments from my gaming stream and create optimized social media content for different platforms.

HIGHLIGHT MOMENTS: [Paste the moments identified by previous prompts]
STREAM TRANSCRIPT CONTEXT: [Paste relevant sections of transcript around each moment]

Create platform-specific content:

## TIKTOK CONTENT (15-60 seconds)
For each highlight, create:
- **Hook Line**: Opening text that grabs attention in first 2 seconds
- **Optimal Length**: Recommended clip duration for maximum engagement
- **Trending Hashtags**: Current gaming/entertainment hashtags to use
- **Text Overlay**: Key phrases to display on screen
- **Sound Strategy**: Whether to use original audio or trending sounds

## YOUTUBE SHORTS (15-60 seconds)
For each highlight, create:
- **SEO Title**: Searchable title that includes game name and action
- **Description**: Brief description with relevant keywords
- **Thumbnail Moment**: Best frame for click-worthy thumbnail
- **End Screen**: Call-to-action for full stream or subscribe
- **Keywords**: Tags for YouTube search optimization

## INSTAGRAM REELS (15-30 seconds)
For each highlight, create:
- **Caption**: Engaging caption that encourages comments
- **Story Integration**: How to tease this content in Instagram Stories
- **Community Hashtags**: Gaming community hashtags for discovery
- **Interactive Elements**: Questions or polls to boost engagement
- **Cross-Promotion**: How to drive traffic to Twitch/YouTube

## TWITTER CLIPS (15-45 seconds)
For each highlight, create:
- **Tweet Text**: Engaging text that works with or without video
- **Thread Opportunity**: Whether this could start a discussion thread
- **Reply Strategy**: How to engage with comments on this post
- **Retweet Value**: Why other gamers would want to share this
- **Community Tags**: Relevant gaming community accounts to mention

Please provide specific, actionable content suggestions that maximize each platform's algorithm and audience preferences.
```
